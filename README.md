# Vicsek_banc_evolution
Simulation 2D de poissons et requins basée sur le modèle de Vicsek.

## Description
- Chaque poisson a un "bruit génétique" qui influence ses mouvements.
- Les requins chassent les poissons les plus isolés.
- Les images sont générées dans le dossier `frames/`.

## Usage
1. Ouvrir le notebook `simulation2D.ipynb`.
2. Modifier les paramètres si nécessaire.
3. Exécuter les cellules du notebook.
4. Exécuter la dernière cellule pour créer un GIF à partir de toutes les images.
